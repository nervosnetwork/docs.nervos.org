# Hardware

Since March 2020, CKB mining has gradually shifted towards ASIC mining. Mining CKB requires purchasing professional mining machines developed by third-party manufacturers.

<img
  alt="mining hardware"
  src={useBaseUrl("img/mining/change-in-ckb-mining.png")}
  width="688"
/>

CKB miners can be acquired from platforms such as: [estripe.com](http://estripe.com/), [Bitmain Official Store](https://shop.bitmain.com/?coin=CKB), [Wyminer](https://www.wyminer.com/), and [Newegg](https://www.newegg.com/).

Platforms like [f2pool](https://www.f2pool.com/miners?currency_code=ckb) offer insights into mining revenue comparison, helping you make informed decisions.
