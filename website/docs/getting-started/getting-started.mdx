# Getting Started

#### Your go-to resources for dApp development on CKB.

## Quick Setup with OffCKB

You can set up your local development environment by installing [@offckb/cli](https://www.npmjs.com/package/@offckb/cli). It provides a one-line command to start a Devnet, pre-funded test accounts
and useful [Scripts (Smart Contracts)](/docs/concepts/glossary#script) like [Omnilock](https://github.com/cryptape/omnilock) and [Spore-contract](https://github.com/sporeprotocol/spore-contract).

### Install

```bash
npm install -g @offckb/cli
```
