To begin, you'll need to install `@offckb/cli` to establish a local dev environment and initialize the project.

```mdx-code-block
<Tabs>
  <TabItem value="offckb-install" label="Command">
```
```bash
npm install -g @offckb/cli
```
```mdx-code-block
  </TabItem>
  <TabItem value="offckb-install-result" label="Response">
```
```bash
changed 106 packages in 17s

11 packages are looking for funding
run `npm fund` for details
```
```mdx-code-block
  </TabItem>
</Tabs>
```

Use `offckb` to select the <code>{props.templateName}</code> template to init the project to your local environment

```mdx-code-block
<Tabs>
  <TabItem value="offckb-init" label="Command">
```
```bash
offckb init <project-name>
```

```mdx-code-block
  </TabItem>
  <TabItem value="offckb-init-result" label="Response">
```
